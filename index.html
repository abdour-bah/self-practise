Introduction

Python is a high-level, easy-to-read programming language used by beginners and professionals alike. It was created by Guido van Rossum and first released in 1991. Python emphasizes code readability and simplicity, making it great for learning how to program. It's widely used in web development, data science, artificial intelligence, automation, and more. Thanks to its large community and extensive libraries, Python continues to grow as one of the most popular programming languages in the world.

Variables

In Python, a variable is used to store data that can be used later in your program. You create a variable by simply assigning a value to a name using the = sign, like x = 5. Python is dynamically typed, which means you don’t need to declare the type of the variable — it figures it out for you. Variable names should be descriptive and follow naming rules (like no spaces and not starting with a number). You can store different types of data in variables, such as numbers, text (strings), lists, and more.

What is a Variable? A variable is a name that refers to a value stored in memory. It acts like a label to store data you want to use or modify later.

Variables Naming Rules

  • Can contain letters, numbers, and underscores

Functions

  • A function is a reusable block of code that performs a specific task.
  • A function is a reusable block of code that performs a specific task.

Loops

  • Loops are used to repeat a block of code multiple times.
  • Python has two main types of loops: for loops and while loops.
  • Loops are powerful for automating repetitive tasks and processing data.
